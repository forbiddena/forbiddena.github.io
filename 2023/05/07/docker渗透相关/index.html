4.利用工具-CDK

https://github.com/cdk-team/CDK

CDK包括三个功能模块

-Evaluate: 容器内部信息收集，以发现潜在的弱点便于后续利用。

-Exploit: 提供容器逃逸、持久化、横向移动等利用方式。

-Tool: 修复渗透过程中常用的linux命令以及与Docker/K8s API交互的命令。

用法：

cdk evaluate [–full]

cdk run –list

cdk run [options]

cdk nc [options]

vi、ps、ifconfig
